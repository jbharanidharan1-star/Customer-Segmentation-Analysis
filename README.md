Customer Segmentation Analysis
📌 Project Overview

This project analyzes customer shopping behavior using Python, Machine Learning, and Power BI.

The project includes:

Data Cleaning
Exploratory Data Analysis
Customer Segmentation using K-Means Clustering
Business Insights
Interactive Dashboard
🛠 Tools Used
Python
Pandas
Scikit-learn
Power BI
PowerPoint

📊 Key Insights

Middle Age & Senior customers contribute highest revenue
Spring season generates maximum sales
New York is the top-performing location
Premium customer clusters identified using K-Means clustering

📈 Dashboard

Interactive Power BI dashboard created for business decision-making.
